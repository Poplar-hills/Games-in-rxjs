# Games-in-rxjs
Simple games that: 
- implemented with [RxJS 5](http://reactivex.io/rxjs/), [Ramda](https://github.com/ramda/ramda) and Canvas
- tested with [AVA](https://github.com/avajs/ava)
- build with [Browserify](http://browserify.org/)
